# Tantata

  - Framework Django
  - DB Squlite3

## Datos generales

- __Entidades definidas__

  - Cuentas
  - Movimientos
  - Personas
  - Periodos

  - Usuarios

  - ABM de Cuentas

  - ABM de Usuarios

  - Mantenimiento de saldos de cuentas
  - Registracion de Depositos
  - Registracion de Cheques
  - Exportacion de reportes
  - Importacion de resumenes por Template
